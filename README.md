# drawing-board
## Sobre o programa
Este é um programa cria uma interface gráfica de usuário através de um quadro onde é possível desenhar de forma livre utilizando como principal a biblioteca **Turtle**.

## Pré-requisitos
- Python 3.11.4
- Turtle
- TKinter
